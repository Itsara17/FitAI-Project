# FitAI: Your Elite Virtual Personal Trainer 🏋️‍♂️✨
> "ยกระดับการดูแลสุขภาพ ด้วยเทรนเนอร์ AI ระดับท็อปคลาสที่เข้าถึงได้ทุกคน"

FitAI คือแอปพลิเคชันที่เปรียบเสมือนตัวแทนพนักงานฟิตเนสระดับพรีเมียม ที่จะคอยดูแลคุณอย่างใกล้ชิดและแม่นยำ โปรเจคนี้พัฒนาขึ้นเพื่อทลายกำแพงข้อจำกัดในการออกกำลังกาย ให้ทุกคนสามารถมีสุขภาพที่ดีได้อย่างปลอดภัยและถูกต้องตามหลักวิทยาศาสตร์การกีฬา

---

## 🚨 Problem Statement (ปัญหาที่เราต้องการแก้)

1. **Financial Barrier (กำแพงด้านราคา):** ค่าบริการ Personal Trainer (PT) ในปัจจุบันมีราคาสูงเฉลี่ย 500-1,000 บาท/ชั่วโมง ทำให้คนทั่วไปเข้าถึงคำแนะนำที่ถูกต้องและเป็นระบบได้ยาก
2. **Injury Risks (ความเสี่ยงจากการบาดเจ็บ):** การฝึก Weight Training โดยปราศจากผู้เชี่ยวชาญดูแล นำไปสู่การใช้ Form (ท่าทาง) ที่ผิดหลักกายวิภาค และการฝึกฝนที่เกินขีดจำกัด (Overtraining) ซึ่งอาจก่อให้เกิดการบาดเจ็บรุนแรง เช่น กล้ามเนื้อฉีกขาด หรือข้อต่อเสื่อมสภาพ
3. **The Nutrition Gap (ช่องโหว่ด้านโภชนาการ):** การสร้างกล้ามเนื้อ (Hypertrophy) หรือลดไขมัน ต้องอาศัยการคำนวณสารอาหารที่แม่นยำ แต่คนส่วนใหญ่มักประสบปัญหา "กินไม่ถึง" หรือคำนวณสัดส่วนโภชนาการไม่ถูกต้อง ทำให้ผลลัพธ์ไม่เกิด หรือเสี่ยงต่อภาวะขาดสารอาหาร

## 🌍 SDG Mapping (เป้าหมายการพัฒนาที่ยั่งยืน)

โปรเจคนี้สอดคล้องกับเป้าหมายขององค์การสหประชาชาติ (UN) ดังนี้:
* **SDG 3 (Good Health and Well-being):** ส่งเสริมสุขภาพที่ดี ลดความเสี่ยงจากโรคภัยไข้เจ็บ และป้องกันการบาดเจ็บจากการออกกำลังกายที่ผิดวิธี
* **SDG 4 (Quality Education):** ทำหน้าที่เป็น "แหล่งเรียนรู้ฟรี" ด้านวิทยาศาสตร์การกีฬาและโภชนาการ เพื่อลดความเหลื่อมล้ำในการเข้าถึงข้อมูลสุขภาพที่มีคุณภาพ

## 🎯 Scope of Project (ขอบเขตการทำงาน)

แอปพลิเคชันจะเน้นที่ **"Personalization" (การปรับแต่งเฉพาะบุคคล)** โดยมีระบบการทำงานหลักดังนี้:

### 1. User Profile Input (ระบบรับข้อมูลผู้ใช้งาน)
* **กายภาพ:** อายุ, เพศ, น้ำหนัก, ส่วนสูง
* **สุขภาพ:** โรคประจำตัว, ประวัติการบาดเจ็บ (เช่น ปวดหลังล่าง, เจ็บเข่า)
* **เป้าหมาย:** ลดน้ำหนัก (Fat Loss), เพิ่มกล้ามเนื้อ (Bulking), หรือรักษาสุขภาพ (Maintenance)

### 2. AI Calculation Engine (ระบบคำนวณอัจฉริยะ)
* **BMI (Body Mass Index):** ดัชนีมวลกายเพื่อประเมินเกณฑ์น้ำหนัก
* **BMR (Basal Metabolic Rate):** พลังงานพื้นฐานที่ร่างกายต้องการขณะพัก
* **TDEE (Total Daily Energy Expenditure):** พลังงานรวมที่ใช้ในแต่ละวันตามระดับ Activity

### 3. Personalized Workout & Nutrition (แผนการฝึกและโภชนาการเฉพาะบุคคล)
* **Smart Workout Plan:** จัดตารางท่าออกกำลังกาย พร้อมกำหนด Reps (จำนวนครั้ง) และ Sets (จำนวนรอบ) ที่เหมาะสมกับเป้าหมาย
* **Safety & Breathing Guide:** แนะนำเทคนิคการหายใจที่ถูกต้อง (เช่น จังหวะออกแรงให้หายใจออก จังหวะผ่อนให้หายใจเข้า) เพื่อป้องกันการบาดเจ็บ
* **Macro Nutrition Tracking:** แนะนำปริมาณ Protein, Carb และ Fat ที่ต้องได้รับต่อวัน เพื่อแก้ปัญหา "กินไม่ถึง"

### 4. Specialized AI Chatbot (แชทบอทผู้เชี่ยวชาญส่วนตัว)
* **Domain-Specific:** ตอบคำถามเฉพาะทางด้านฟิตเนสและโภชนาการได้อย่างแม่นยำ กระชับ และตรงประเด็น
* **Emotion & Sentiment Detection:** ตรวจจับอารมณ์ของผู้ใช้ (เช่น ท้อแท้, มีไฟ) เพื่อปรับโทนการตอบสนองให้กำลังใจได้อย่างเหมาะสม
* **Customizable Character:** สามารถกำหนดบุคลิกของเทรนเนอร์ AI ได้ (เช่น โทนดุดันแบบทหาร หรือ โทนใจดีให้กำลังใจ)
* **Bilingual Support:** รองรับการสนทนาทั้งภาษาไทยและภาษาอังกฤษอย่างเป็นธรรมชาติ
* **Premium UI/UX:** ออกแบบกราฟิกสวยงาม ใช้งานง่าย มอบประสบการณ์เสมือนใช้บริการฟิตเนสระดับพรีเมียม

## 🗄️ Data Architecture (สถาปัตยกรรมข้อมูล)

| ประเภทข้อมูล (Data Selection) | แหล่งที่มา (Data Collection) | วัตถุประสงค์การใช้งาน |
| :--- | :--- | :--- |
| **User Data** (น้ำหนัก, ส่วนสูง, ประวัติบาดเจ็บ, เป้าหมาย) | Input จากผู้ใช้งานบนแอปพลิเคชัน | ประมวลผลเพื่อสร้าง Personalization Plan |
| **Exercise Data** (ท่าฝึก, แคลอรี่ที่เผาผลาญ, ระดับความยาก) | Open Datasets จาก **Kaggle** (เช่น Exercise & Fitness Metrics) | นำมาเป็นฐานข้อมูลในการจัดตารางฝึก |
| **Nutrition Data** (แคลอรี่, Macros: Protein/Carb/Fat) | API มาตรฐาน เช่น **Nutritionix API** | คำนวณและแนะนำโภชนาการให้ตรงกับเป้าหมาย |
| **Anatomy & Safety Data** (ข้อควรระวัง, ชีวกลศาสตร์) | แหล่งอ้างอิงที่น่าเชื่อถือ (**WHO**, กรมอนามัย, งานวิจัยกีฬา) | เป็นขอบเขตความปลอดภัย (Guardrails) ป้องกันการบาดเจ็บ |
| **Conversational Base** (บทสนทนาและความรู้ทั่วไป) | **Gemini API** / **OpenAI API** | ขับเคลื่อนแชทบอทให้ตอบคำถามเป็นธรรมชาติและถูกต้อง |
